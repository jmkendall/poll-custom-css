# poll-custom-css
custom css tests for poll.ma.pe 

URL example:
https://poll.ma.pe/overlay/<APPID>?customCss=<URL to css file in github>

URL example with greenscreen:
https://poll.ma.pe/overlay/<APPID>?greenscreen=true&customCss=<URL to css file in github>


to update an option with a different game, update the url in the css file
url('https://cdn.cloudflare.steamstatic.com/steam/apps/<APPID>/header.jpg')

The easiest way to find the APPID is to go to the game's page on Steam and copy the number from the URL.
